Przygotowanie przed warsztatami

Przejdź te kroki spokojnie, w swoim tempie. Zajmą maksymalnie kilkanaście minut. Dzięki temu na warsztatach od razu będziemy mogli pracować z agentem Claude Code. Odhaczaj po kolei, a w razie wątpliwości pytaj na grupie.

Twój postęp zapisuje się w tej przeglądarce. Możesz zamknąć stronę i wrócić później.

1

Kup płatną subskrypcję Claude

Narzędzie, którego użyjemy na warsztatach (Claude Code), działa na płatnym koncie Claude. Wystarczy najtańszy plan Pro. Nie potrzebujesz żadnego „klucza API" ani planu firmowego.

  1. Wejdź na claude.com/pricing.
  2. Wybierz plan Pro (plan Max też zadziała, ale na start nie jest potrzebny).
  3. Załóż konto lub zaloguj się i opłać subskrypcję.
  4. Zapamiętaj e-mail i hasło. Tym samym kontem zalogujesz się później w VS Code.
Otwórz cennik Claude →

To samo konto Claude zaloguje Cię potem we wtyczce w VS Code. Nie płacisz drugi raz.

2

Pobierz i zainstaluj VS Code

VS Code to darmowy edytor od Microsoftu. To w nim będziesz rozmawiać z Claude. Kliknij duży przycisk pobierania, strona sama wykryje Twój system.

Pobierz VS Code →
Na Macu
  1. Kliknij przycisk pobierania (strona sama wybierze wersję dla Maca).
  2. Otwórz pobrany plik. Pojawi się ikona Visual Studio Code.
  3. Przeciągnij tę ikonę do folderu Aplikacje (Applications).
  4. Uruchom VS Code z Launchpada lub folderu Aplikacje. Jeśli Mac zapyta „czy na pewno chcesz otworzyć", kliknij Otwórz.
Na Windowsie
  1. Kliknij przycisk pobierania (wersja dla Windows).
  2. Uruchom pobrany plik VSCodeUserSetup...exe.
  3. Zaakceptuj warunki i klikaj Dalej, zostaw domyślne ustawienia.
  4. Kliknij Zainstaluj, a na końcu Zakończ (możesz zaznaczyć „Uruchom VS Code").

Potrzebna wersja 1.98 lub nowsza. Świeżo pobrany VS Code zawsze ją ma, więc nie musisz nic sprawdzać.

3

Zrób na pulpicie folder o nazwie agent

To zwykły folder, jak każdy inny na komputerze.

  • Mac: kliknij prawym przyciskiem na pulpicie → Nowy folder.
  • Windows: kliknij prawym przyciskiem na pulpicie → Nowy → Folder.

Nazwij go dokładnie: agent, małymi literami, bez spacji.

4

Otwórz folder agent w VS Code

Uruchom VS Code. Z górnego menu wybierz File → Open Folder... (po polsku Plik → Otwórz folder...), wskaż folder agent z pulpitu i kliknij Otwórz.

Jeśli VS Code zapyta, czy ufasz autorom w tym folderze, kliknij Yes, I trust the authors (Tak, ufam autorom).

5

Dodaj rozszerzenie Claude Code

  1. Naciśnij Cmd+Shift+X (Mac) lub Ctrl+Shift+X (Windows), żeby otworzyć panel Extensions (Rozszerzenia) po lewej.
  2. W pole wyszukiwania na górze wpisz claude code for vs i wybierz pierwszą pozycję.
  3. Upewnij się, że to rozszerzenie od wydawcy Anthropic, i kliknij Install.
  4. Jeśli po instalacji pojawi się przycisk Enable (Włącz), kliknij go, żeby aktywować wtyczkę.

Wtyczka ma w sobie wszystko, czego trzeba. Nie musisz otwierać terminala ani niczego doinstalowywać.

6

Zaloguj się swoim kontem Claude

  1. Poszukaj ikony Spark (iskra): jest na lewym pasku (Activity Bar) i w prawym górnym rogu edytora. Kliknij ją, żeby otworzyć panel Claude Code.
  2. Kliknij Sign in.
  3. Otworzy się przeglądarka. Zaloguj się tym samym kontem Claude, na którym wykupiłeś subskrypcję, i potwierdź.
  4. Wróć do VS Code. Panel pokaże, że jesteś zalogowany. Może pojawić się checklista Learn Claude Code, spokojnie możesz ją zamknąć.

Nie widzisz ikony Spark? Zamknij i otwórz VS Code ponownie, powinna się pojawić.

7

Napisz pierwszą wiadomość do Claude

W panelu Claude Code, na dole, jest pole tekstowe. Wpisz w nie:

Panel Claude Code
> cześć, czy mnie słyszysz? Enter

Naciśnij Enter. Jeśli Claude odpowie, wszystko działa i jesteś gotowy na warsztaty.

Zainstaluj Obsidian

Obsidian to darmowy program do notatek w zwykłych plikach tekstowych. Przyda się, żeby trzymać w jednym miejscu pomysły, prompty i materiały, które potem podsuwasz Claude.

  1. Wejdź na obsidian.md i kliknij Download (strona sama wykryje Twój system).
  2. Zainstaluj tak samo jak VS Code: na Macu przeciągnij do folderu Aplikacje, na Windowsie uruchom pobrany instalator.
Pobierz Obsidian →

Zainstaluj Wispr Flow

Wispr Flow zamienia Twój głos na tekst w dowolnym polu, także w rozmowie z Claude. Zamiast wystukiwać długie polecenia, po prostu je mówisz. Działa na Macu i Windowsie.

  1. Kliknij przycisk poniżej (mój link), pobierz aplikację dla swojego systemu i zainstaluj.
  2. Uruchom Wispr Flow i przejdź krótką konfigurację mikrofonu.
Pobierz Wispr Flow →

Pobranie przez ten link daje 30 dni okresu próbnego zamiast standardowych 14. To mój link polecający.

Załóż darmowe konto na GitHubie

GitHub to miejsce w sieci, gdzie przechowujesz i wersjonujesz swoje projekty. Na warsztatach zrobisz pierwszy „push", czyli wyślesz swój folder do GitHuba. Wystarczy darmowe konto, załóż je wcześniej, żeby nie tracić na to czasu na żywo.

  1. Wejdź na github.com/signup.
  2. Podaj e-mail, wymyśl hasło i nazwę użytkownika (username).
  3. Potwierdź konto, klikając link w mailu, który przyśle Ci GitHub.

Zainstaluj wtyczkę Office Viewer

Office Viewer to darmowe rozszerzenie do VS Code. Dzięki niemu otworzysz w edytorze pliki Worda, Excela, PowerPointa i PDF-y, a pliki tekstowe .md zobaczysz ładnie sformatowane, nie jako surowy zapis. Przyda się, bo w folderze agenta często lądują właśnie takie dokumenty.

Instalujesz je tak samo jak wtyczkę Claude Code, z poziomu VS Code:

  1. W VS Code naciśnij Cmd+Shift+X (Mac) lub Ctrl+Shift+X (Windows), żeby otworzyć panel Extensions (Rozszerzenia) po lewej.
  2. W pole wyszukiwania wpisz office viewer.
  3. Wybierz pozycję Office Viewer (niebieska ikona z literą „A", wydawca Database Client) i kliknij Install.
